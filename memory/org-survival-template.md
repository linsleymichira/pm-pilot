# Org Survival Map

_Power map of key stakeholders. Updated after significant interactions. Political intelligence — not a people directory._

---

## How to Use This

Before any high-stakes meeting, decision, or communication:
1. Find the relevant person's entry
2. Note what they want and what risks they carry
3. Check for active tensions involving them
4. Adjust your approach accordingly

This file is read automatically by `/meeting-prep` for person-based prep.
